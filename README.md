# ton-nft-collection-snapshot

## Setup

Install dependency:

```bash
npm install
```

Create file `.env`

```ini
NFT_COLLECTION=<ADDRESS>
```

Optional create and add [toncenter.com API key](https://toncenter.com/):

```ini
TONCENTER_KEY=<KEY>
```

## Run

Run on testnet:

```bash
npm test
```

Run on mainnet:

```bash
npm start
```
